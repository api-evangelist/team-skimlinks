# Team SkimlinksThis is a repo for managing machine readable artifacts for the [Team Skimlinks APIs](http://skimlinks.com/skimlinks). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Team Skimlinks APIs](http://skimlinks.com/skimlinks).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://skimlinks.com/skimlinks)- [Developer](http://skimlinks.com/skimlinks)- [Twitter](https://twitter.com/skimlinks)- [GitHub](https://github.com/Skimlinks)The [APIs.json](https://github.com/api-evangelist/team-skimlinks/blob/master/apis.json) index provides links to all the relevant links for the Team Skimlinks APIs, including the OpenAPI and Postman if they are present.
